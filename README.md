# theon

A new and featured way to build generic HTTP API clients.

## Features

- Modular pluggable design
- Built-in middleware layer

## Installation

Via npm (node.js):
```bash
npm install theon --save
```

Via bower:
```bash
bower install theon --save
```

Or loading the script directly:
```html
<script src=""></script>
```
